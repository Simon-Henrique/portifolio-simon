Sistema Cadastro Python

Sistema feito em python que, com uma interface gráfica amigável e intuitiva, é capaz de registrar os inputs feito pelo usuário em um banco de dados, no exemplo, foi utilizado o SQL Server. Esse conceito pode ser aplicado a uma infinidade de soluções. Utiliza os pacotes tkinter, pyodbc.

Cotação de Moedas

Uma aplicação feita em python que consome uma API com os valores atualizados das principais moedas atualizadas a cada 30 segundos. Essa aplicação importa 3 pacotes, tkinter (interface_gráfica), pillow(manipulação de imagem) e requests(requisição get).

Catalogo de jogos

Este é um desafio batido! O desafio pedia para criar um catálogo de jogos. O usuário pode preencher o catálogo a partir de um formulário e ver os jogos que já estão preenchidos. Além disso, o usuário pode pesquisar um jogo e filtrar a lista. Este projeto foi feito usando angular, typescript, javascript, html, css. Este projeto consome dados de uma API, a versão final está usando um backend hospedado no supabase.

Covid Cases

Projeto com origem de um desafio, tem como objetivo apresentar os casos de covid pelo mundo e suas variantes através de um mapa mundi. Projeto elaborado com Angular 12, também faz uso da lib amChart 5 para a renderização do mapa, os dados são consumidos através de uma API rest hospedada no supabase. Banco de dados pode ficar em standby periodicamente.

Fundamentos de React

Projeto que apresenta através de cards, a resolução de pequenos desafios utilizando conceitos básicos de React

Pokedex

Projeto de uma pokedex utilizando as tecnologias relacionadas ao angular: HTML5, SASS, JavaScript, TypeScript e Angular. Esta aplicação gera, de forma dinâmica e responsiva, os cards de cada pokemon através de uma lista de objetos, esta lista de objetos é obtida através do RESTful pokemon API.

Lista de Tarefas

Lista de tarefas feita em angular, javascript, typescript,html e scss. Este projeto também utiliza o conceito de CRUD, entretanto, dessa vez mais modularizado, de forma que cada regra de negócio está em um módulo individual. Esta aplicação consome uma API elaborada especificamente para este estudo que serve como back-end.

Sistema CRUD

Uma aplicação de um cadastro feita em angular, com o conceito CRUD. Este projeto faz uso dos principais recursos do angular como observables, diretivas, services, rotas, propert binding, pipes etc. Esta aplicação frontend também faz interação com um backend simples, feito através de um json server de forma que o angular possa consumir uma API.

Cálculo IMC

Este projeto foi desenvolvido com javascript puro, possui validações de dados, o calculo IMC em sí, e condicionais simples para respostas correspondente a cada range de peso.

Analisador de Números

Este projeto foi desenvolvido com javascript puro, possui validação para receber apenas números de 1 à 100 e que ainda não foram inclusos na variável, ao finalizar a aplicação realiza cálculos utilizando os dados números inputados pelo usuário.

Verificador de idade

Projeto feito em javascript onde é considerado em uma variável a data atual (do sistema) e a data de nascimento preenchida pelo usuário e é feito o cálculo da idade atual. Possui validação e condicionais.

Cronômetro

Cronômetro simples elaborado a partir do método setInterval() do JS.

Bom dia!

Sistema capaz de identificar o horário atual do sistema através do recurso JS getHours() e através de condicionais simples, a página retorna resultados visuais conforme horário.
